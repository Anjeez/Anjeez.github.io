Free word order

_a- context phrase
_e- subject
_i- verb
_o- object
_u- indirect object

Words modifying each other get prefixed with the same thing.
Every word in a modifying verb phrase get the prefixes put after the main prefix
A sentence in the context phrase has every word prefixed with it an _a- prefix
Multiple context phrases/subjects/verbs/objects/indirect objects has the prefix start with different consonants

a|bkʃd
e|fghx
i|ʒlmn
o|prst
u|vwjz

If a consonant cluster happens, or if a consonant and the following vowel does not make a valid prefix, then it's a start of a root.
