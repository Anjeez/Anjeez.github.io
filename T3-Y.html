B3+Y: Thanks for Playing my game but each pair of identical tiles will be played once

Type: String Curse

Personal Curse Name: Oncer

Edited from: Each tile will be played twice

If there are two tiles in a row with the same angle, then the second tile is removed.

12-Y: Artificial Chariot but the acute angles and the obtuse angles are swapped

Type: String Curse

Personal Curse Name: ObAcu

From Yangsy's website: The angle of each tile is now its supplementary angle. Integer multiples of 180° will bypass this curse, while angles >180° will be moduled by 180° when processing. This effect can also add or removing twirls without changing the rhythm for better readability.
